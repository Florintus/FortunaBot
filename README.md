# FortunaBot 🎉

Telegram бот для проведения розыгрышей с проверкой подписок.

## Функционал

- Создание розыгрышей с настройкой даты и времени публикации
- Проверка подписок на Telegram каналы
- Проверка подписок на Twitch каналы
- Инлайн-календарь для выбора дат
- Автоматическая публикация и подведение итогов
- Уведомление победителей в личные сообщения
- Панель управления для администраторов

## Технологии

- Python 3.11
- pyTelegramBotAPI (telebot)
- SQLAlchemy + PostgreSQL
- APScheduler
- Docker / Docker Compose
